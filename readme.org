A loose set of extensions for Michael Edward's [[https://github.com/mdedwards/slippery-chicken][slippery-chicken]].

In order to use one of the files, include them via
~(load "/path/to/file.lsp")~.

Details on /slippery-chicken/ can be found [[https://github.com/mdedwards/slippery-chicken][here]].
